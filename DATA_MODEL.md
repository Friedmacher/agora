# Agora — Data Model

> **Version:** 1.0 | **Status:** AUTHORITATIVE

---

## Entity-Relationship Diagram

```
admin_users (1) ──< refresh_tokens (N)      [token rotation]
admin_users (1) ──< visits (N)              [visit ownership]
locations   (1) ──< visits (N)
locations   (1) ──< images (N)
```

All entities use `uuid` PKs (PostgreSQL `gen_random_uuid()`).

---

## `images`

| Column | Type | Constraints | Notes |
|---|---|---|---|
| id | uuid | PK | |
| location_id | uuid | FK → locations NOT NULL ON DELETE CASCADE | |
| storage_key | varchar(500) | NOT NULL | MinIO object path: `{location_id}/{uuid}.{ext}` |
| display_order | smallint | NOT NULL DEFAULT 0 | Lower = shown first |
| created_at | timestamptz | NOT NULL DEFAULT now() | |

> **No `url` column.** Presigned URLs are generated by the API at serve time and never persisted (see `ARCHITECTURE.md` Section 4.4).

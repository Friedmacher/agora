# Agora
Agora is a personal hospitality location manager.
An admin maintains a curated log of visited places (Periplus) and a wish-list of places to visit (Pothos).
A map view (Pinax) displays all places geographically.
A public read-only view of Periplus, Pothos, and Pinax is accessible to anyone without authentication. The admin dashboard (Tholos) is restricted to authenticated admin users.
